# Carey Academic Helper

## Overview
JHU Carey operates on a non-standard 8-week term system. This skill ensures academic planning remains accurate by programmatically checking the official Carey calendar data.

## When to use
- To identify if a date falls within Term 1 or Term 2.
- To check for academic breaks (e.g., Spring Break) that differ from the main JHU campus.
- To provide a countdown to the 2026 Commencement.

## Technical Details
- Input: `{"date": "YYYY-MM-DD"}`
- Script: `scripts/carey_engine.py`
