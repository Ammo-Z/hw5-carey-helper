# HW5: Agent Skill - Carey Academic Helper

## Purpose
This skill provides deterministic answers regarding the JHU Carey Business School 2025-2026 academic calendar.

## Why a script?
Carey's 8-week term structure is complex. LLMs often hallucinate dates for "Term 2" or "Reading Days". This script acts as a "source of truth" to prevent those errors.
